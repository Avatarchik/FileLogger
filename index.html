FileLogger Feature Overview

FileLogger is a logging tool for Unity.

Log to a file

Set output file name or use default log.txt and all your logging will be saved to hard drive.

Open log file in external log viewer

Use an external log viewer, like the free LogExpert to search, filter and highlight the data the way you want!
